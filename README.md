# Uncharted Wiki

Una wiki interactiva dedicada a la saga Uncharted, creada con React y Vite. Explora juegos, personajes, historia y más sobre las aventuras de Nathan Drake y Chloe Frazer.

## 🚀 Características

- **Exploración de Juegos**: Detalles completos de cada título de la saga Uncharted.
- **Interfaz Responsiva**: Diseño adaptable a móviles y computadoras.
- **Modo Oscuro/Claro**: Cambia entre temas para una experiencia personalizada.
- **Modal de Compra**: Enlaces directos a PlayStation Store y Steam para adquirir los juegos.
- **Navegación Intuitiva**: Menú desplegable con dropdown para la saga completa.

## 🛠️ Tecnologías Utilizadas

- **React**: Framework para la interfaz de usuario.
- **Vite**: Herramienta de build rápida y moderna.
- **CSS**: Estilos personalizados con variables CSS para temas.
- **React Router**: Navegación entre páginas.
- **ESLint**: Linting para mantener la calidad del código.

## 📦 Instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/uncharted-wiki.git
   cd uncharted-wiki
   ```

2. Instala las dependencias:
   ```bash
   npm install
   ```

3. Ejecuta el servidor de desarrollo:
   ```bash
   npm run dev
   ```

4. Abre [http://localhost:5174](http://localhost:5174) en tu navegador.

## 🚀 Build para Producción

Para generar una versión optimizada para producción:

```bash
npm run build
```

Los archivos se generarán en la carpeta `dist`.

## 📱 Uso

- Navega por las secciones usando el menú superior.
- Cambia entre modo claro y oscuro con el botón de tema.
- Haz clic en "Comprar" para acceder a opciones de compra en PlayStation Store o Steam.
- Explora páginas individuales de juegos para más detalles.
